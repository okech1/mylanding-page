FarmShare

Connecting farmers and landowners with ease.

Key Features

User Registration and Login

Secure user authentication using JWT tokens. Users can register and log in securely.

Listing Creation

Create listings for farm equipment and land with images and detailed descriptions.

Search and Filter

Find listings based on location, type of equipment, and availability.

About FarmShare

FarmShare was inspired by the need to connect farmers and landowners easily and securely. The idea came about during a brainstorming session where we identified the challenges faced by small-scale farmers in accessing equipment and land. Our goal is to bridge this gap and create a community where resources are shared efficiently.

This project is a Portfolio Project for Holberton School.
